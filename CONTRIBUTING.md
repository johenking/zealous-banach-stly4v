# Contributing

Thanks for helping improve Grade Planner.

## Good first contributions

- Add curriculum examples for more degree programs
- Improve mobile accessibility and keyboard navigation
- Add tests for GPA, credit, and completion calculations
- Improve documentation and screenshots
- Report bugs with reproduction steps

## Local development

```bash
npm install
cp .env.example .env.local
npm start
```

## Pull requests

Please keep pull requests focused and include:

- A short description of the user problem
- Screenshots for UI changes
- Notes about Firebase or data model changes
- Test/build results when relevant
